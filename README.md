**Live Demo:** [https://nagraj-hrms.netlify.app/](https://nagraj-hrms.netlify.app/)

---

## 📌 Project Overview

**NAGRAJ HRMS** is a comprehensive **role-based workforce management system** built with the MERN stack. It streamlines the entire employee lifecycle—from worker registration to final onboarding—through a multi-level approval workflow.

> **Core Idea:** Workers register themselves, then the system routes their applications through **Contractor → Production → CEO → HR**, with each role having specific access and approval rights. HR finally **finalizes, accepts, or rejects** workers and generates official documents like **joining letters**.

---

## 🎯 Key Features & Workflow

### 👥 Multi-Level Role-Based Access

| Role | Responsibilities |
|------|------------------|
| **Worker** | Self-registration, profile management, view application status |
| **Contractor** | Initial screening, document verification, forward to Production |
| **Production** | Skill assessment, technical evaluation, forward to CEO |
| **CEO** | Final strategic approval/rejection |
| **HR** | **Finalize** workers, generate joining letters, manage employee records |
| **Admin** | System oversight, role management, master data control |
